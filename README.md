# Setup Files for PyCharm
A script to install the Astral tools (uv, ruff, ty) and a skelton TOML

## Example use

- within terminal cd to PyCharm new project root directory
- curl -sL https://raw.githubusercontent.com/amuir2k/python-template/main/setup_dev.sh | bash
- chmod +x setup_dev.sh
- ./setup_dev.sh
- Edit project name in TOML to match project
